SquadMaster

SquadMaster est un organizer Dofus Unity open source pour Windows, pensé pour les joueurs multicompte. Il permet de gérer les fenêtres, les personnages, les équipes, les raccourcis globaux et les invitations de groupe. C’est une alternative moderne à Dofus Organizer et nAiO Organizer, développée par Appsgard.

Repérez les clients Unity, désignez un chef de groupe, envoyez des invitations (/invite) et pilotez vos fenêtres — même lorsque l’interface est réduite dans la zone de notification.

Dernière version — exécutable Windows 64 bits (self-contained, runtime .NET inclus), publié sur GitHub Releases.

Prérequis : Windows 10 ou 11 (64 bits). L’exe de release est prêt à l’emploi (aucun SDK requis pour l’installer). À quoi ça sert ?

SquadMaster automatise ce qui est répétitif en multicompte Dofus Unity : gestion des fenêtres, focus, chat, raccourcis et invitations de groupe.

Gestion des fenêtres

Détection des clients Dofus Unity, extraction du nom de personnage depuis le titre de fenêtre, tableau trié (chef en tête, puis initiative).

Invitations de groupe

Le personnage « chef » envoie les /invite Nom dans le jeu, avec des temporisations et des profils de vitesse pour s’adapter à votre machine.

Raccourcis & équipes

Touches globales (afficher l’app, rafraîchir, ré-inviter, précédent/suivant par Team 1 / Team 2) et raccourci par personnage pour activer une fenêtre.

Configuration & support

Paramètres dans %AppData%\SquadMaster\, import optionnel des touches depuis dofus.json, journaux et paquet support (ZIP) en cas de souci.

Premier lancement

Résumé du flux recommandé — l’application intègre aussi une aide détaillée.

  1. Lancez Dofus (une fenêtre par personnage) et connectez les personnages.
  2. Ouvrez SquadMaster, cliquez Rafraîchir : les lignes apparaissent avec les noms détectés.
  3. Cochez Chef de groupe sur le personnage qui doit envoyer les invitations dans le chat.
  4. Cochez Sélection sur les personnages à inviter (en général tous sauf le chef).
  5. Utilisez Inviter (ou le raccourci global configuré). En cas d’échec partiel, essayez un profil de vitesse plus lent ou vérifiez la touche d’ouverture du chat (import Dofus).

Une alternative moderne à Dofus Organizer et nAiO Organizer

Beaucoup de joueurs Dofus multicompte ont utilisé ou recherchent encore Dofus Organizer, nAiO ou nAiO Organizer pour gérer leurs fenêtres, raccourcis et personnages. SquadMaster répond au même besoin pour Dofus Unity, avec une application Windows récente, open source et pensée pour les usages actuels du multicompte.

SquadMaster n’est pas affilié à Ankama, Dofus Organizer ou nAiO. Il s’agit d’un outil indépendant développé par Appsgard.

Questions fréquentes

SquadMaster est-il un Dofus Organizer ?

Non. SquadMaster est un outil indépendant pour Dofus Unity. Il aide à gérer les fenêtres, les raccourcis, les personnages, les équipes et les invitations en multicompte — une alternative moderne, pas un remplaçant officiel de Dofus Organizer.

SquadMaster remplace-t-il nAiO Organizer ?

SquadMaster peut répondre aux mêmes usages courants (fenêtres, raccourcis, personnages, équipes) sur Dofus Unity. C’est une alternative récente et open source, sans lien avec nAiO ou nAiO Organizer.

SquadMaster fonctionne-t-il avec Dofus Unity ?

Oui. SquadMaster détecte les clients Dofus Unity ouverts sur Windows et s’appuie sur le titre des fenêtres pour identifier les personnages connectés.

SquadMaster est-il gratuit ?

Oui. SquadMaster est open source et gratuit. Le code source et les releases sont disponibles sur GitHub.

SquadMaster est-il lié à Ankama ?

Non. SquadMaster n’est pas affilié à Ankama. C’est un outil indépendant développé par Appsgard pour faciliter le multicompte sur Dofus Unity.

Fermeture de l’application

Pour garder les raccourcis globaux actifs, préférez masquer SquadMaster dans la zone de notification plutôt que de quitter complètement.

Les raccourcis sont enregistrés via l’API Windows : ils ne fonctionnent que tant que le processus tourne. Référez-vous à la boîte de dialogue de comportement à la fermeture dans l’app pour choisir masquer ou quitter.
