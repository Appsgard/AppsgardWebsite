SquadMaster

SquadMaster est un outil Windows open source pour le multicompte Dofus Unity, pensé comme une alternative moderne à Dofus Organizer. Repérez les clients Unity, désignez un chef de groupe, envoyez des invitations de groupe (/invite) et gérez les fenêtres avec des raccourcis globaux — même lorsque l’interface est réduite dans la zone de notification.

Dernière version — exécutable Windows 64 bits (self-contained, runtime .NET inclus), publié sur GitHub Releases.

Toutes les releases · Code source · Discord

Prérequis : Windows 10 ou 11 (64 bits). L’exe de release est prêt à l’emploi (aucun SDK requis pour l’installer). Pour compiler le dépôt vous-même : SDK .NET 8 avec la charge de travail développement Desktop (WPF). Au premier lancement, un antivirus peut être plus curieux avec ce type de binaire self-contained — comportement fréquent.

À quoi ça sert ?

SquadMaster automatise ce qui est répétitif en multicompte Dofus Unity : gestion des fenêtres, focus, chat, raccourcis et invitations de groupe.

Gestion des fenêtres

Détection des clients Dofus Unity, extraction du nom de personnage depuis le titre de fenêtre, tableau trié (chef en tête, puis initiative).

Invitations de groupe

Le personnage « chef » envoie les /invite Nom dans le jeu, avec des temporisations et des profils de vitesse pour s’adapter à votre machine.

Raccourcis & équipes

Touches globales (afficher l’app, rafraîchir, ré-inviter, précédent/suivant par Team 1 / Team 2) et raccourci par personnage pour activer une fenêtre.

Configuration & support

Paramètres dans %AppData%\SquadMaster\, import optionnel des touches depuis dofus.json, journaux et paquet support (ZIP) en cas de souci.

Premier lancement

Résumé du flux recommandé — l’application intègre aussi une aide détaillée.

  1. Lancez Dofus (une fenêtre par personnage) et connectez les personnages.
  2. Ouvrez SquadMaster, cliquez Rafraîchir : les lignes apparaissent avec les noms détectés.
  3. Cochez Chef de groupe sur le personnage qui doit envoyer les invitations dans le chat.
  4. Cochez Sélection sur les personnages à inviter (en général tous sauf le chef).
  5. Utilisez Inviter (ou le raccourci global configuré). En cas d’échec partiel, essayez un profil de vitesse plus lent ou vérifiez la touche d’ouverture du chat (import Dofus).

Fermeture de l’application

Pour garder les raccourcis globaux actifs, préférez masquer SquadMaster dans la zone de notification plutôt que de quitter complètement.

Les raccourcis sont enregistrés via l’API Windows : ils ne fonctionnent que tant que le processus tourne. Référez-vous à la boîte de dialogue de comportement à la fermeture dans l’app pour choisir masquer ou quitter.
